预算建议

每次安装成本（eCPI）分级规则

  • 核心口径：优先使用 AppsFlyer 广告日报返回的官方 eCPI。
  • 低量级：最近 3 天激活量 < 15，默认观察，不自动调价。
  • 中量级：最近 3 天激活量 15-30。eCPI ≤ 目标 90% 时上调 20%，≥ 目标 115% 时下调 20%。
  • 高量级：最近 3 天激活量 > 30。eCPI ≤ 目标时上调 20%，≥ 目标 120% 时下调 20%。
  • 目标 eCPI：优先按同应用、同平台、同媒体源的有效关键词 P40 计算，再用同平台中位数做上限钳制。
  • 极端低效且已进入 pause_candidate 的关键词，才会给出暂停建议。

这是给调控师参考的预算动作建议。系统不会自动执行，需要人工确认后再处理。 每次安装成本（eCPI）
